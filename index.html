The Unofficial
Cursor API

Cursor is a great editor — but it ships without the standard API endpoints you expect from a model provider. This fills that gap: drop-in /v1/chat/completions and /v1/responses, authenticated with your own Cursor key.

Free to use for now — while it isn’t costing us much to run.

Setup

use standard SDKs with your Cursor Subscription

Four steps from a Cursor key to streaming completions.

  1. 1

    Get a Cursor API key

    Sign in at cursor.com/dashboard, open Integrations → API Keys, and create a key. It looks like key_…. Treat it like a password — this site forwards it straight to Cursor and never stores it.

  2. 2

    Use the base URL and Bearer token

    Send Authorization: Bearer <your-cursor-api-key> to either endpoint below.

  3. 3

    OpenAI SDK

    Set baseURL and apiKey. Chat Completions still works everywhere; the Responses API is what OpenAI now recommends for new projects.

  4. 4

    Vercel AI SDK

    Create a provider with createOpenAI, then pick openai.responses(…) or openai.chat(…) and pass it to streamText.
