Endpoint update

The hosted API is moving out of the request path.

Cursor asked us to take down the hosted API endpoint. We are keeping the current routes online temporarily while this local app is verified, but the production path is now the local macOS app.

The local app is the better long-term experience: your API key stays on your machine, agent tools run against your real project folders, and OpenCode, Codex, or Factory can connect to a normal OpenAI-compatible localhost URL.
